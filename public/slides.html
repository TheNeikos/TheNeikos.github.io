Zabbix

Übersicht

Zabbix ist eine Open Source Applikation zum Monitoring von Netzwerken sowie Applikationen.

Es besteht aus einer Server Komponente und mehreren Clienten die Daten an den Server schicken, dieser Analysiert sie und agiert basierend auf Regeln.

Konkretes Beispiel

Freier Speicher im letztem Monat

Informationen

  • 1.0 veröffentlicht 2004 unter GPL
  • 2.2 veröffentlicht 2013
  • Wird von einer Firma mit gleichem Namen weiter-entwickelt

Platformen

  • Server funktioniert nur auf Linux
  • Clients gibt es für Linux, Solaris, {Free, Open}BSD, OS X, Window >2003

Info Quellen

  • Jeder Agent (Client) gibt Daten an den Server frei
  • Der Server fragt diese ab und speichert sie
